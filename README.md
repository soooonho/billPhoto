## 📖 프로젝트 소개 (The Legend Begins...)

태초에 빈 캔버스(IDE)가 있었습니다. 그리고 5명의 요리사(개발자)가 모였습니다.
각자의 철학과 고집, 그리고 **'일단 돌아가면 건드리지 마'** 정신이 하나로 뭉쳐, 전무후무한 전설의 스파게티 코드 `billPhoto`가 탄생했습니다.

이 코드는 단순한 프로그램이 아닙니다. 읽는 이의 멘탈을 붕괴시키고, 종국에는 해탈에 이르게 하는 한 편의 포스트모더니즘 문학입니다.

## 👨‍🍳 5인의 스파게티 요리사 (The Spaghetti Chefs)

이 엄청난 요리를 완성한 주역들입니다.

* 🍝 **Chef 1. [우상이]** - *들여쓰기 파괴자 (The Indenter)*
    * "Tab과 Space는 섞어 써야 제맛이죠."
* 🍝 **Chef 2. [이진우]** - *전역 변수 성애자 (Global Master)*
    * "변수는 역시 어디서나 접근 가능해야 열린 사회 아닙니까?"
* 🍝 **Chef 3. [곽민준]** - *복붙의 닌자 (Ctrl+C & Ctrl+V Ninja)*
    * "StackOverflow는 신이고, 나는 무적이다."
* 🍝 **Chef 4. [권순호]** - *매직 넘버 술사 (The Magic Number)*
    * `if (x == 7482)` - "왜 7482냐고요? 묻지 마세요. 바꾸면 서버 터집니다."
* 🍝 **Chef 5. [김다한]** - *주석 증발러 (No Comment)*
    * "내 코드는 너무 완벽해서 주석 따윈 필요 없다. (나도 3일 뒤엔 못 읽음)"

## 🚀 조리 방법 (How to Run)

1. 리포지토리를 눈을 감고 클론합니다.
   ```bash
   git clone [https://github.com/soooonho/billPhoto.git](https://github.com/soooonho/billPhoto.git)
   ```
2. 해당 디렉토리로 이동합니다.
   ```bash
   cd billPhoto
   ```

2. 패키지를 설치하며 무사히 돌아가길 빕니다.
   ```bash
   npm install  # 또는 pip install -r requirements.txt 등
   ```
3. 실행합니다. (에러 로그가 떠도 당황하지 마세요. 그것도 요리의 일부입니다.)
   ```bash
   npm run start
   ```
